Luogu P1972 解题报告

题目大意

询问区间中有多少个不同的数字。

思路

这道题有树状数组/线段树的做法，这里说一下一种优秀的暴力算法——莫队算法。

莫队算法的精髓，是将一些问题同时处理，以优化暴力。

当可以通过ans[l, r]来推出ans[l, r + 1], ans[l + 1, r], ans[l, r - 1], ans[l - 1, r]时，你就可以使用莫队算法啦~

简单的来说，莫队算法将所有的询问按照原序列分类，一般每n\sqrt{n}√​n​​​个元素分为一个块。注意，也可以不按照n\sqrt{n}√​n​​​来分，但要注意时间复杂度的问题。

将左端点所在块相同的放在一起，按照右端点不降的顺序排序。

来分析一下时间复杂度：

左端点会在n\sqrt{n}√​n​​​个块内移动，每次移动不超过n\sqrt{n}√​n​​​，而右端点则会最多移动n个点，所以整体的时间复杂度是O(nn)O(n\sqrt{n})O(n√​n​​​)的。
