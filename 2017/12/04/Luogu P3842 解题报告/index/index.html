Luogu P3842 解题报告

题目大意

每行给你一条线段，求一条最短路径，使得从(1,1)走到(n,n)，且完全经过所有线段的最短距离。

注意：只能向左、右、上走。

思路

f[i][0/1]f[i][0/1]f[i][0/1]表示第i行，走完之前的线段并且走完当前行线段后停留在左端点还是右端点。 即有： f[i][0]=min(f[i][0],f[i−1][0]+abs(b[i]−a[i−1])+d);f[i][0] = min(f[i][0], f[i - 1][0] + abs(b[i] - a[i - 1]) + d);f[i][0]=min(f[i][0],f[i−1][0]+abs(b[i]−a[i−1])+d); f[i][0]=min(f[i][0],f[i−1][1]+abs(b[i]−b[i−1])+d);f[i][0] = min(f[i][0], f[i - 1][1] + abs(b[i] - b[i - 1]) + d);f[i][0]=min(f[i][0],f[i−1][1]+abs(b[i]−b[i−1])+d); f[i][1]=min(f[i][1],f[i−1][0]+abs(a[i]−a[i−1])+d);f[i][1] = min(f[i][1], f[i - 1][0] + abs(a[i] - a[i - 1]) + d);f[i][1]=min(f[i][1],f[i−1][0]+abs(a[i]−a[i−1])+d); f[i][1]=min(f[i][1],f[i−1][1]+abs(a[i]−b[i−1])+d);f[i][1] = min(f[i][1], f[i - 1][1] + abs(a[i] - b[i - 1]) + d);f[i][1]=min(f[i][1],f[i−1][1]+abs(a[i]−b[i−1])+d);

其中d=abs(a[i]−b[i])+1;d = abs(a[i] - b[i]) + 1;d=abs(a[i]−b[i])+1;，即当前行长度并且加上走行的距离。

最后不要忘记求到(n,n)的距离。
